#  Task API (Flask)

API REST desenvolvida em Python para gerenciamento de tarefas, com operações completas de CRUD.

---

##  Sobre o projeto

Este projeto foi criado para praticar:

- Criação de APIs REST
- Integração com banco de dados
- Organização de código 
- Boas práticas com Flask

---

##  Tecnologias utilizadas:

- Python
- Flask
- SQLAlchemy
- SQLite

---

## ⚙️ Funcionalidades

- ✅ Criar tarefas
- ✅ Listar tarefas
- ✅ Atualizar tarefas
- ✅ Deletar tarefas

---

## ▶️ Como rodar o projeto

```bash
# Para clonar o repositório:
git clone https://github.com/rqueiroga/task-api-flask.git

# Para entrar na pasta:
cd task-api-flask

# Para criar ambiente virtual:
python -m venv venv

# Para ativar ambiente:
venv\Scripts\activate   # Windows
source venv/bin/activate # Linux/Mac

# Para instalar dependências:
pip install -r requirements.txt

# Para rodar a aplicação:
python app.py
